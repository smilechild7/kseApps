오늘의 성경 말씀 개인정보 처리방침

최종 업데이트: 2025년 10월 12일

App Store에서 앱 보기 프로젝트 개요

1. 수집하는 정보

  • 사용자가 입력한 고민/기도 제목 등의 텍스트 (최대 50자)
  • 언어 설정 및 위젯 상태와 같은 앱 환경 정보
  • 앱과 위젯이 최근에 추천한 성경 구절의 식별자
  • 광고 제공을 위한 기기 광고 식별자(Advertising ID/IDFA) 및 기본 광고 이벤트

2. 사용 목적

  • 입력된 텍스트를 기반으로 맞춤형 성경 구절을 추천하기 위해 OpenAI API에 전달합니다.
  • 추천 결과를 앱과 위젯에서 표시하고, 동일한 구절이 반복되지 않도록 히스토리를 관리합니다.
  • 언어 및 설정 정보를 저장하여 사용자 경험을 유지합니다.
  • Google AdMob을 통해 광고를 노출하고, 부정 클릭 방지 및 성과 분석을 수행합니다.

3. 보관 및 파기

최근 추천 정보와 언어 설정은 App Group UserDefaults에 저장되며, 사용자가 앱을 삭제하면 함께 삭제됩니다. OpenAI API 호출 시 전달된 데이터는 응답 생성 목적에만 사용되며, 앱 내에서 별도로 장기 보관하지 않습니다.

4. 데이터 공유

  • 사용자 입력 텍스트는 맞춤 추천 생성을 위해 OpenAI Responses API에 전송됩니다.
  • 광고 제공을 위해 Google AdMob SDK가 기기 광고 식별자 및 기본 광고 성과 데이터를 수집할 수 있습니다.
  • 이 외의 제3자에게는 개인 정보를 제공하지 않습니다.

5. 이용자의 권리

  • 앱을 삭제하면 저장된 로컬 데이터가 모두 제거됩니다.
  • 기기 설정에서 맞춤형 광고 추적 제한(광고 추적 제한/앱 추적 투명성)을 설정할 수 있습니다.
